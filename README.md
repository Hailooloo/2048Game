# 2048 数字合成小游戏

一个使用原生 HTML / CSS / JavaScript 实现的 2048 小游戏，零依赖、纯静态，开箱即用。

![游戏截图](screenshot.png)

## ✨ 特性

- **流畅合并动画**：方块滑动、合并缩放、新块弹出，配合 `+N` 得分上浮提示
- **最高分记录**：通过 `localStorage` 持久化保存，刷新或关闭页面都不丢失
- **撤销功能**：支持回退上一步（最多 25 步历史）
- **多端操作**：桌面端方向键，移动端滑动手势
- **响应式设计**：自适应手机与桌面屏幕
- **现代 UI**：暖色调主题、毛玻璃遮罩、浮动光斑背景与动效按钮

## 🎮 操作说明

| 操作 | 桌面端 | 移动端 |
| --- | --- | --- |
| 移动方块 | 方向键 `↑` `↓` `←` `→` | 滑动屏幕 |
| 撤销 | 按键 `Z` 或点击「撤销」 | 点击「撤销」 |
| 重新开始 | 按键 `R` 或点击「新游戏」 | 点击「新游戏」 |

合并相同数字的方块，目标是合成出 **2048**！

## 🚀 本地运行

无需任何构建工具，直接用浏览器打开 `index.html` 即可：

```bash
# 方式一：直接双击 index.html

# 方式二：起一个本地静态服务器（推荐，避免个别浏览器限制）
python3 -m http.server 8080
# 然后访问 http://localhost:8080
```

## 🌐 部署到 GitHub Pages

1. 将本项目推送到 GitHub 仓库
2. 进入仓库 `Settings → Pages`
3. Source 选择 `main` 分支、根目录 `/`
4. 保存后即可通过 `https://<你的用户名>.github.io/<仓库名>/` 在线访问
